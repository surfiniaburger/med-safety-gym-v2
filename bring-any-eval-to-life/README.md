# Run and deploy your AI Studio app

This contains everything you need to run your app locally.

View your app in AI Studio: https://ai.studio/apps/bundled/bring_any_idea_to_life

## Features

- **Artifact Selection Engine**: Browse and select clinical safety evaluation results directly from GitHub.
- **The Gauntlet**: A 3D neural pathway visualization that maps model rewards and safety violations in real-time.
- **On-Demand Simulation**: Instantly transform safety failures into interactive clinical training scenarios using Gemini.
- **Cinematic Controls**: Dynamic camera profiles and manual speed adjustments for detailed data exploration.

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
